## Gardening App 
Android application developed with Kotlin and Room Database in order to store plant collections and trigger watering reminders.

### Technologies:
- Kotlin
- Room Database
- MVVM
- Camera 
- Coil
- Alarm Manager
- Android Animated Theme Manager
